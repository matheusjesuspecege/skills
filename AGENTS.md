# AGENTS.md — Linka Eventos Platform

## Sobre o Projeto

O **Linka Eventos** é uma plataforma digital de gestão de eventos desenvolvida pelo Pecege. Surgiu em 2020 durante a pandemia de Covid-19 para adaptar atividades presenciais para o formato remoto e evoluiu para um produto próprio com o objetivo de:

- Reduzir custos com plataformas externas
- Tornar-se uma potencial fonte de receita
- Oferecer experiência personalizada para organizadores e participantes

Funcionalidades atuais incluem criação de landing pages, controle de acesso, dashboard de inscrições e integrações com sistemas externos.
